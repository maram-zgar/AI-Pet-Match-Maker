Une application intelligente qui recommande l’animal idéal à adopter en fonction de la personnalité, du mode de vie et des préférences de l’utilisateur.
Le système utilise un chatbot interactif, du traitement du langage naturel, et un modèle de machine learning (KNN) pour proposer des correspondances fiables et personnalisées.

Présentation du projet

Le nombre d’animaux abandonnés dans les refuges ne cesse d’augmenter. Pour assurer une adoption responsable, il est essentiel de proposer un animal vraiment compatible avec la personnalité et le mode de vie de l’adoptant.
L’objectif de Pet Match-Maker est d’offrir une expérience d’adoption intelligente, interactive et personnalisée.

L’utilisateur répond à un chatbot (texte ou voix), et l’application recommande l’animal le plus adapté grâce à un algorithme de machine learning.

Objectifs

Faciliter le processus d’adoption.
Offrir une interaction naturelle grâce au traitement du langage et à la voix.
Automatiser la sélection des animaux via un algorithme de recommandation fiable.
Fournir une interface simple, moderne et intuitive.

Architecture générale

Frontend : Interface Flask (HTML/CSS/JS)
Backend : Python
NLP : spaCy pour extraire les caractéristiques utilisateur
ML : KNN avec scikit-learn
Base de données fictive : générée avec Faker
Images : PlaceCats, PlaceDog
